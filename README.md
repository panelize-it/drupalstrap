Drupalstrap
===========

A distribution for Drupal 7 built with Twitter Bootstrap and Panels.

  - Custom views
    - Portfolio
    - Clients
    - Blog
    - Accordion Item
    - Tabs Item
    - Carousel Item

  - Custom content types
    - Portfolio
    - Clients
    - Blog
    - Accordion Item
    - Tabs Item
    - Carousel Item
    - Page
